# 位运算

查看一个整数二进制第k位是几(0/1):**(n >> k) & 1**,注意优先级顺序.

lowbit(x):返回最后一个1及该1后面所有的0,如101000,lowbit(10100)返回100.

```cpp
//lowbit(x)实现
int lowbit(x)
{
    return x &-x;
}

```
